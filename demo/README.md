# Demo 演示脚本目录

## 📁 目录说明

本目录包含了项目的各种演示脚本，用于展示特定功能和使用方法。

### 📋 演示脚本列表

| 脚本名 | 功能描述 | 适用场景 |
|--------|----------|----------|
| **demo_prompts.py** | 提示词演示 | 展示LLM提示词构建和脚本生成 |
| **demo_test_cleanup.py** | 测试清理演示 | 展示测试产物自动清理功能 |

### 🚀 使用方式

#### 1. 提示词演示
```bash
# 运行提示词演示
python demo/demo_prompts.py

# 查看提示词构建过程
python demo/demo_prompts.py --verbose
```

**功能展示**：
- 新闻脚本提示词构建
- 研究脚本提示词生成
- SSML格式化处理
- 多种脚本风格支持

#### 2. 测试清理演示
```bash
# 运行测试清理演示
python demo/demo_test_cleanup.py

# 查看清理过程详情
python demo/demo_test_cleanup.py --verbose
```

**功能展示**：
- 测试成功后的自动清理
- 测试失败时的产物保留
- 强制保留模式演示
- 清理状态反馈
