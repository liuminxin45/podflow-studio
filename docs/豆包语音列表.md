# Doubao TTS 音色列表（本地备份）

> 来源：官网体验中心
>
> https://www.volcengine.com/product/tts

## 注

1. 下述音色列表中，中文音色可支持中英文混合场景。
2. 病弱少女、活泼女孩、和蔼奶奶、邻居阿姨四个音色暂不支持双向流式接口调用。
3. moon 系列音色，会有概率出现音色跳变/突变问题。
4. 少量 mars 系列音色，会有稳定性问题，如「磁性解说男生（jieshuonansheng）」。
