## 🎯 目录功能说明

### 🚀 **demo/** - 演示脚本
- **用途**: 展示项目功能和使用方法
- **内容**: 提示词演示、测试清理演示
- **特点**: 可视化过程、详细注释、友好输出

### 📚 **docs/** - 项目文档
- **用途**: 完整的项目文档和用户指南
- **内容**: 项目说明、功能指南、参考文档
- **特点**: 中文文档、结构清晰、易于导航

### ⚙️ **env/** - 环境配置
- **用途**: 统一管理环境配置和依赖
- **内容**: 配置文件、虚拟环境、检查工具
- **特点**: 安全管理、版本控制、便于部署

### 💻 **src/** - 源代码
- **用途**: 核心业务逻辑实现
- **内容**: 各功能模块的源代码
- **特点**: 模块化设计、清晰架构、易于扩展

### 🛠️ **工具脚本**
- **run.py**: 主程序入口
- **check_env.py**: 环境检查工具
- **run_tests.py**: 测试运行器
- **clean_lint_cache.py**: 缓存清理工具

## 🎯 使用指南

### 🚀 **快速开始**
```bash
# 1. 检查环境配置
python check_env.py

# 2. 运行主程序
python run.py --help

# 3. 查看演示
python demo/demo_prompts.py
```

### 📖 **学习路径**
1. **新手**: 查看 `docs/项目说明.md`
2. **开发者**: 查看 `docs/测试清理指南.md`
